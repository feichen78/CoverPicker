## ✅ 已完成功能（当前版本）

### 基础框架与界面
- [x] 迁移至 PySide6（Qt for Python）桌面应用
- [x] 综合布局：左侧视频列表 + 右侧主工作区（分段导航 + 截图网格 + 操作工具栏）
- [x] 视频扫描：支持 `Z:\` 目录下常见视频格式（`.mp4`, `.mkv`, `.avi` 等）
- [x] 视频信息显示：时长、大小（分辨率/帧率为预估）
- [x] 分段导航：A/B/C/D/E 五分区切换（基于视频时长自动计算）
- [x] 网格密度切换：9/12/16/25 张布局（16 为默认）
- [x] 排除区间：通过对话框手动设置排除时间段（应用于截图生成）

### 截图生成与显示
- [x] 异步调用 FFmpeg 提取关键帧
- [x] 截图自动裁剪片头片尾（跳过比例 15%）
- [x] 截图网格自适应显示（保持宽高比，不拉伸）
- [x] 每张截图显示时间戳（左下角）
- [x] 选中状态：蓝色边框 + 左上角蓝色圆点
- [x] 锁定状态：金色边框（锁定后不会被刷新/重抽替换）

### 交互操作
- [x] 单击截图：选中/取消选中（支持多选）
- [x] 双击截图：打开放大预览窗口（滚轮缩放、拖动平移）
- [x] 全选 / 取消全选
- [x] 锁定 / 解锁选中截图
- [x] 收藏 / 取消收藏选中截图（右上角 ★ 标记）
- [x] 刷新未锁定（重新生成未锁定的截图，保留锁定项）
- [x] 全部重抽（清除锁定，随机重新采样生成）
- [x] 导出选中的截图到 `StillPic/视频名/` 目录（导出后右下角绿点标记）

### 状态与信息
- [x] 左侧面板实时显示：锁定数、收藏数
- [x] 进度指示：显示当前生成进度（移至左侧视频信息下方）
- [x] 已浏览分区统计（粗略：每加载一个分区标记为已浏览）

### Zoom 精修
- [x] 双击截图触发 Zoom 窗口
- [x] L1-L4 层级切换（每次 9 张，基于选定中心点 ±4s）
- [x] 全局探索模式
- [x] 在 Zoom 窗口中可收藏、锁定、导出
- [x] 状态同步回主网格

## 🚧 未完成功能（按 PRODUCT.md 优先级）

### 🔴 高优先级（核心缺失）
- [ ] **状态持久化（SQLite）**（产品文档 7）  
  所有操作状态（锁定、收藏、导出、已浏览）仅存内存，重启丢失。
- [ ] **视频预览与自由选段**（产品文档 6.6）  
  完全未实现，需集成视频播放器、时间滑块、自由选段随机截图。
- [ ] **视频片段导出**（产品文档 6.5）  
  仅实现了截图导出，无损视频片段导出（FFmpeg stream copy）未实现。
- [ ] **目录选择、搜索、导入、清理缓存**（产品文档 6.7）  
  当前硬编码 `Z:\`，无搜索，导入/清理不可用。

### 🟡 中优先级（提升体验）
- [ ] 自定义分区数量（v1.2 规划）
- [ ] 批量导出全部收藏截图
- [ ] 操作撤销/重做（产品原则要求）
- [ ] 精确视频信息读取（分辨率、帧率等通过 FFprobe 获取）

### 🟢 低优先级（优化完善）
- [ ] 进度条（替代文本提示）
- [ ] 界面主题切换
- [ ] 缓存自动清理策略
