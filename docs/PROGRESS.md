核心交互
主界面布局（左-中-右），视频列表、分区按钮、截图网格、底部操作栏。

网格延迟加载（QListView + 自定义委托），大幅降低内存占用。

单击选中/取消选中，双击放大预览（滚轮缩放 + 拖动平移）。

全选/取消全选、键盘快捷键（Ctrl+A, Ctrl+D, Delete, Space, 方向键）。

分区切换、密度切换（9/12/16/25）。

排除区间设置（支持编辑已有区间、持久化、跨设备同步）。

收藏、锁定、导出、刷新未锁定、全部重抽。

后台与性能
异步 FFmpeg 帧提取（asyncio + 信号量控制并发）。

缓存复用（1 秒内时间差匹配）。

固定缓存目录（%TEMP%/CoverPicker_cache），退出不删除。

缓存统计（大小、文件数）及手动清理。

任务取消即时响应（切换分区时 kill 子进程）。

视频加载时界面立即更新（显示占位图、视频名称提前显示）。

数据持久化
SQLite 数据库存储视频、分区、收藏、排除区间。

跨设备状态同步（通过 file_id 识别）。

自动备份（关闭窗口时备份至指定目录）。

崩溃报告生成（异常捕获 + 日志轮转）。

辅助功能
预览窗口（视频帧预览、时间轴选段、自定义分区、片段导出）。

智能监控（目录变化自动导入、定时扫描、删除检测）。

导入视频（文件/文件夹，基于文件名去重）。

批量删除视频（仅删除数据库记录，不删除文件）。
