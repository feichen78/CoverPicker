## ✅ 已完成功能（当前版本）

### 基础框架与界面
- [x] 迁移至 PySide6（Qt for Python）桌面应用
- [x] 综合布局：左侧视频列表 + 右侧主工作区（分段导航 + 截图网格 + 操作工具栏）
- [x] 视频扫描：支持 `Z:\` 目录下常见视频格式（`.mp4`, `.mkv`, `.avi` 等）
- [x] 视频信息显示：时长、大小（分辨率/帧率为预估）
- [x] 分段导航：A/B/C/D/E 五分区切换（基于视频时长自动计算）
- [x] 网格密度切换：9/12/16/25 张布局（9 为默认）
- [x] 排除区间：通过对话框手动设置排除时间段（已实现但实际未生效）
- [x] **分段状态标记**：已浏览 ✓、有收藏 ★、有导出 ▼ 显示在分段按钮上

### 截图生成与显示
- [x] 异步调用 FFmpeg 提取关键帧
- [x] 截图自动裁剪片头片尾（跳过比例 15%）
- [x] 截图网格自适应显示
- [x] 每张截图显示时间戳（左下角）
- [x] 选中状态：蓝色边框
- [x] 锁定状态：🔒 标记（左上角）
- [x] **跨视频状态缓存**：切换视频时保留锁定/收藏/导出状态（内存中）

### 交互操作
- [x] 单击截图：选中/取消选中（支持多选）
- [x] 双击截图：打开放大预览窗口
- [x] 锁定 / 解锁选中截图
- [x] 收藏 / 取消收藏选中截图（⭐ 标记）
- [x] 刷新未锁定（重新生成未锁定的截图，保留锁定项）
- [x] 全部重抽（清除锁定，随机重新采样生成）
- [x] 导出选中的截图到 `StillPic/视频名/` 目录（导出后绿点标记）

### 状态与信息
- [x] 左侧面板实时显示：锁定数、收藏数
- [x] 进度指示：显示当前生成进度（左侧视频信息下方）
- [x] 分段状态标记：已浏览 ✓、有收藏 ★、有导出 ▼
