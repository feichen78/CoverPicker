# CoverPicker

## 产品定位

CoverPicker 是一个帮助用户从NAS存储的海量视频中快速挑选视频剧照的工具。

目标：

把原本需要花几十分钟挑剧照的工作，

缩短到几分钟完成。

它不是播放器。

它不是剪辑软件。

它只做一件事：

帮助用户找到最满意的一张或几张剧照。
